# frontend-examples

This repository contains small, minimalistic examples and demos of implementations of various interface features.

## О репозитории

Данный репозиторий содержит множество маленьких и независимых проектов, связанных с фронтенд-разработкой. Каждый проект является примером использования или реализации одного или нескольких, связанных наборов фич.

### Примеры тем проектов

- Авторизация и аутентификация
- Подключение Web3 кошелька
- Прохождение KYC
- Подключение и настройка Sentry
- Работа с блокчейном Solana
- Реализация сложных анимаций
- И многое другое...

## Структура репозитория

Каждый проект размещается в отдельной папке в корне репозитория. Все проекты независимы друг от друга и могут быть запущены отдельно.

## Руководство для контрибьюторов

Перед созданием нового проекта ознакомьтесь с [руководством по оформлению проектов](./PROJECT_GUIDELINES.md).
